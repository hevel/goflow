Object view

This is a default template used by the generic application view_object.

You may define a suitable template (the default is view_object.html) with the template parameter in the urls.py file as follows: 

    (r'^my/appli/view_appli/$',
      'goflow.apptools.views.view_object',
      {'template':'mytemplate.html'}),
you can use directly the url provided by goflow.apptools.urls, and define parameters in the activity. This template can access the following variables:
  • object: object (linked to the process instance, instance.wfobject())
  • action_values: list of action values for OK
  • cancel_value: action value for redirect and quit
