Put here a function that returns a user, given a workitem as parameter (more parameters with default values may be added).
This function will be searched in the module defined in settings.WF_PUSH_APPS_PREFIX

example:

in settings.py 

WF_PUSH_APPS_PREFIX = 'leavedemo.leave.pushapplications'

in pushapplications.py 

def route_to_customer(workitem):
    return workitem.instance.user
